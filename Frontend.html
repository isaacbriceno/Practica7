¿Qué es Frontend?

Llamamos frontend a la parte del sitio web con la que el usuario interactúa. Es lo que algunos suelen llamar “el lado del cliente. es el encargado de llevar a la práctica el diseño de un sitio web para que el usuario final pueda usarlo.

    ¿Cuales son sus fundamentos?

  • HTML: Sirve para definir la estructura y el contenido de una página Web.
  • CSS: Es utilizado para describir la presentación de documentos HTML o XML.
  • JavaScript: Es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web.
